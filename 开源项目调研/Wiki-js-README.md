## Overview

Wiki.js is a modern, lightweight and powerful wiki app built on NodeJS.

### Key Features

- **Modern UI**: Beautiful, responsive interface
- **Markdown Support**: Full markdown editing and rendering
- **Multi-database**: Support for PostgreSQL, MySQL, MariaDB, MSSQL, and SQLite
- **Authentication**: Multiple authentication methods (LDAP, OAuth, etc.)
- **Asset Management**: Image and file uploads
- **Access Control**: Granular permissions and page restrictions
- **Editor**: Visual editor and markdown code editor
- **Localization**: Support for 40+ languages
- **API**: RESTful API for automation
- **Theme System**: Customizable themes

### Tech Stack

- **Backend**: Node.js
- **Frontend**: Vue.js
- **Database**: PostgreSQL, MySQL, MariaDB, MSSQL, SQLite
- **License**: AGPLv3
