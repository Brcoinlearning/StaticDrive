# PocketBase 文件上传文档

## 概述

PocketBase 提供了完整的文件存储和管理功能，支持多种文件类型，包括图片、文档、视频等。文件可以直接上传到 PocketBase 服务器或外部存储（如 S3）。

---

## 文件字段类型

在 PocketBase 中，可以通过管理面板或 API 创建文件类型的字段：

### 字段配置选项

- **字段名**: 字段的标识符
- **字段标签**: 显示名称
- **最大文件大小**: 单个文件的最大大小限制
- **允许的文件类型**: 限制可上传的文件扩展名
- **多文件**: 是否允许上传多个文件
- **必填**: 是否要求必须上传文件

---

## 文件上传方式

### 1. 通过管理面板上传

1. 登录 PocketBase 管理面板 (默认: `http://localhost:8090/_/`)
2. 选择集合
3. 创建或编辑记录
4. 点击文件上传区域选择文件
5. 保存记录

### 2. 通过 JavaScript SDK 上传

#### 单文件上传

```javascript
import PocketBase from 'pocketbase';

const pb = new PocketBase('http://localhost:8090');

// 创建 FormData
const formData = new FormData();

// 添加文本字段
formData.append('title', 'My Document');
formData.append('description', 'This is a test document');

// 添加文件
const fileInput = document.getElementById('fileInput');
formData.append('file', fileInput.files[0]);

// 上传
const record = await pb.collection('documents').create(formData);

console.log(record);
```

#### 多文件上传

```javascript
const formData = new FormData();
formData.append('title', 'Multiple Files');

// 添加多个文件 (使用相同的字段名)
for (let i = 0; i < fileInput.files.length; i++) {
  formData.append('files', fileInput.files[i]);
}

const record = await pb.collection('documents').create(formData);
```

#### 更新文件

```javascript
// 更新记录的文件字段
const formData = new FormData();
formData.append('file', newFile);

const updated = await pb.collection('documents').update(
  'RECORD_ID',
  formData
);
```

### 3. 通过 REST API 上传

#### cURL 示例

```bash
# 上传单个文件
curl -X POST http://localhost:8090/api/collections/documents/records \
  -F "title=My Document" \
  -F "file=@/path/to/document.pdf"

# 上传多个文件
curl -X POST http://localhost:8090/api/collections/documents/records \
  -F "title=Multiple Files" \
  -F "files=@/path/to/file1.pdf" \
  -F "files=@/path/to/file2.jpg"

# 更新文件
curl -X PATCH http://localhost:8090/api/collections/documents/records/RECORD_ID \
  -F "file=@/path/to/new-document.pdf"
```

#### Fetch API 示例

```javascript
const formData = new FormData();
formData.append('title', 'Upload via Fetch');
formData.append('file', fileInput.files[0]);

const response = await fetch(
  'http://localhost:8090/api/collections/documents/records',
  {
    method: 'POST',
    body: formData
  }
);

const record = await response.json();
```

## 文件存储位置

### 本地存储

默认情况下，文件存储在以下目录结构中：

```
/pb_data/
  └── storage/
      └── {collection_name}/
          └── {record_id}/
              └── {field_name}/
                  └── {file_name}
```

示例：
```
/pb_data/storage/documents/abc123_xyz789/thumbnail/sample.jpg
```

### S3 存储

PocketBase 支持将文件直接存储到 AWS S3 或兼容服务：

#### 配置 S3

1. 在管理面板中进入 **Settings > S3**
2. 填写以下信息：
   - **Bucket**: S3 存储桶名称
   - **Region**: AWS 区域
   - **Endpoint**: 自定义端点（用于 S3 兼容服务）
   - **Access Key**: AWS 访问密钥
   - **Secret**: AWS 密钥
   - **Force Path Style**: 是否使用路径样式

#### 测试 S3 连接

在管理面板中点击 **Test S3 connection** 验证配置。

## 文件访问

### 获取文件 URL

文件上传后，PocketBase 会自动生成文件 URL：

```javascript
const record = await pb.collection('documents').getOne('RECORD_ID');

// 文件 URL 格式
// http://localhost:8090/api/files/{collection}/{id}/{field_name}/{file_name}
console.log(record.file); // 文件对象或 URL
```

### 通过 API 访问文件

```bash
# 直接访问文件
curl http://localhost:8090/api/files/documents/RECORD_ID/file/sample.pdf

# 获取文件缩略图 (如果是图片)
curl http://localhost:8090/api/files/documents/RECORD_ID/file/sample.jpg?token=THUMB_TOKEN
```

### JavaScript 显示文件

```javascript
// 图片文件
const imageUrl = pb.files.getUrl(record, record.avatar);

// 文档文件
const fileUrl = pb.files.getUrl(record, record.document);

// HTML 中使用
document.getElementById('myImage').src = imageUrl;
```

## 图片处理

PocketBase 自动为图片生成多个尺寸的缩略图：

### 可用尺寸

- **原始**: 完整尺寸
- **thumb**: 80x80px
- **small**: 200x200px
- **medium**: 400x400px
- **large**: 800x800px

### 使用缩略图

```javascript
// 获取缩略图 URL
const thumbUrl = pb.files.getUrl(record, record.image, {
  thumb: 'thumb' // 'thumb', 'small', 'medium', 'large'
});
```

### 裁剪模式

```javascript
// 填充模式 (默认，裁剪以适应)
pb.files.getUrl(record, record.image, { thumb: 'small' });

// 适应模式 (不裁剪)
pb.files.getUrl(record, record.image, {
  thumb: 'small',
  scaleCover: false
});
```

## 文件删除

### 删除单个文件

```javascript
// 清空文件字段
const updated = await pb.collection('documents').update('RECORD_ID', {
  file: null
});
```

### 删除整个记录

删除记录时会自动删除关联的文件：

```javascript
// 删除记录及其文件
await pb.collection('documents').delete('RECORD_ID');
```

### REST API 删除

```bash
# 清空文件字段
curl -X PATCH http://localhost:8090/api/collections/documents/records/RECORD_ID \
  -H "Content-Type: application/json" \
  -d '{"file": null}'

# 删除记录
curl -X DELETE http://localhost:8090/api/collections/documents/records/RECORD_ID
```

## 文件验证

### 客户端验证

```javascript
function validateFile(file) {
  // 检查文件大小
  const maxSize = 5 * 1024 * 1024; // 5MB
  if (file.size > maxSize) {
    alert('文件太大！最大 5MB');
    return false;
  }

  // 检查文件类型
  const allowedTypes = ['image/jpeg', 'image/png', 'application/pdf'];
  if (!allowedTypes.includes(file.type)) {
    alert('不支持的文件类型！');
    return false;
  }

  return true;
}
```

### 服务端验证

在集合设置中配置：
1. 进入集合的 **Edit collection**
2. 编辑文件字段
3. 设置：
   - **Max upload size**: 最大文件大小
   - **Allowed file types**: 允许的扩展名（如：`.jpg,.png,.pdf`）

## 文件上传示例

### 完整的上传表单示例

```html
<!DOCTYPE html>
<html>
<head>
  <title>文件上传示例</title>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/pocketbase/0.21.1/pocketbase.umd.min.js"></script>
</head>
<body>
  <form id="uploadForm">
    <input type="text" id="title" placeholder="标题">
    <input type="file" id="file" multiple>
    <button type="submit">上传</button>
  </form>

  <div id="progress"></div>
  <div id="result"></div>

  <script>
    const pb = new PocketBase('http://localhost:8090');

    document.getElementById('uploadForm').addEventListener('submit', async (e) => {
      e.preventDefault();

      const formData = new FormData();
      formData.append('title', document.getElementById('title').value);

      const files = document.getElementById('file').files;
      for (let i = 0; i < files.length; i++) {
        formData.append('files', files[i]);
      }

      try {
        const record = await pb.collection('documents').create(formData);
        document.getElementById('result').innerHTML = `
          <p>上传成功！</p>
          <p>记录 ID: ${record.id}</p>
        `;
      } catch (error) {
        document.getElementById('result').innerHTML = `
          <p>上传失败: ${error.message}</p>
        `;
      }
    });
  </script>
</body>
</html>
```

### React 上传组件示例

```jsx
import React, { useState } from 'react';
import PocketBase from 'pocketbase';

const pb = new PocketBase('http://localhost:8090');

function FileUpload() {
  const [uploading, setUploading] = useState(false);
  const [error, setError] = useState(null);

  const handleUpload = async (event) => {
    event.preventDefault();
    setUploading(true);
    setError(null);

    try {
      const formData = new FormData();
      formData.append('title', event.target.title.value);

      const files = event.target.files.files;
      for (let i = 0; i < files.length; i++) {
        formData.append('files', files[i]);
      }

      await pb.collection('documents').create(formData);
      alert('上传成功！');
    } catch (err) {
      setError(err.message);
    } finally {
      setUploading(false);
    }
  };

  return (
    <form onSubmit={handleUpload}>
      <input name="title" placeholder="标题" />
      <input name="files" type="file" multiple />
      <button type="submit" disabled={uploading}>
        {uploading ? '上传中...' : '上传'}
      </button>
      {error && <p>{error}</p>}
    </form>
  );
}
```

const formData = new FormData();

### 上传进度追踪

```javascript
// 使用 XMLHttpRequest 获取进度
const xhr = new XMLHttpRequest();

xhr.upload.onprogress = (e) => {
  if (e.lengthComputable) {
    const percentComplete = (e.loaded / e.total) * 100;
    console.log(`上传进度: ${percentComplete}%`);
  }
};

xhr.open('POST', 'http://localhost:8090/api/collections/documents/records');
xhr.send(formData);
```

### 大文件分块上传

对于非常大的文件，考虑在前端实现分块上传：

```javascript
async function uploadInChunks(file, chunkSize = 5 * 1024 * 1024) {
  const chunks = Math.ceil(file.size / chunkSize);

  for (let i = 0; i < chunks; i++) {
    const start = i * chunkSize;
    const end = Math.min(start + chunkSize, file.size);
    const chunk = file.slice(start, end);

    // 上传分块
    await uploadChunk(chunk, i, chunks);
  }
}
```

## 安全考虑

1. **认证**: 确保上传端点需要适当的认证
2. **文件大小限制**: 设置合理的文件大小限制
3. **文件类型验证**: 验证文件的实际类型，而不仅仅是扩展名
4. **病毒扫描**: 考虑为上传的文件添加病毒扫描
5. **存储位置**: 生产环境使用 S3 等专用存储服务

---

## 故障排除

### 常见问题

1. **文件太大**
   - 检查集合设置中的最大文件大小
   - 检查 PocketBase 服务的 body limit 配置

2. **文件类型不支持**
   - 在集合设置中添加文件扩展名

3. **S3 上传失败**
   - 验证 S3 凭证
   - 检查存储桶权限
   - 确保存储桶存在

4. **文件无法访问**
   - 检查文件权限
   - 验证 URL 格式
   - 确认记录存在且未被删除
