# PocketBase 扩展机制文档

## 概述

PocketBase 提供了强大的扩展机制，允许开发者通过 Go 代码、Hooks、自定义路由和 JavaScript VM 来扩展功能。这使得 PocketBase 既可以用作开箱即用的后端，也可以作为可定制的后端框架。

---

## 扩展方式概览

PocketBase 支持以下扩展方式：

1. **Hooks（钩子）** - 在特定事件发生时执行自定义代码
2. **Custom Routes（自定义路由）** - 添加自定义 API 端点
3. **Go Embedding（Go 嵌入）** - 将 PocketBase 嵌入到自定义 Go 应用中
4. **JavaScript VM (JSVM)** - 使用 JavaScript 编写扩展逻辑
5. **Migrations（迁移）** - 数据库结构迁移
6. **Plugins（插件）** - 可重用的扩展包

## 1. Hooks（钩子）

Hooks 允许你在特定的生命周期事件中执行自定义代码。

### 可用的 Hook 类型

#### OnModelBeforeCreate / OnModelAfterCreate
```go
// 在创建记录之前
app.OnModelBeforeCreate().Add(func(e *core.ModelEvent) error {
    fmt.Printf("即将创建记录: %v\n", e.Model)
    return nil
})

// 在创建记录之后
app.OnModelAfterCreate().Add(func(e *core.ModelEvent) error {
    fmt.Printf("已创建记录: %v\n", e.Model)
    return nil
})
```

#### OnModelBeforeUpdate / OnModelAfterUpdate
```go
// 在更新记录之前
app.OnModelBeforeUpdate().Add(func(e *core.ModelEvent) error {
    fmt.Printf("即将更新记录: %v\n", e.Model)
    return nil
})

// 在更新记录之后
app.OnModelAfterUpdate().Add(func(e *core.ModelEvent) error {
    fmt.Printf("已更新记录: %v\n", e.Model)
    return nil
})
```

#### OnModelBeforeDelete / OnModelAfterDelete
```go
// 在删除记录之前
app.OnModelBeforeDelete().Add(func(e *core.ModelEvent) error {
    fmt.Printf("即将删除记录: %v\n", e.Model)
    return nil
})

// 在删除记录之后
app.OnModelAfterDelete().Add(func(e *core.ModelEvent) error {
    fmt.Printf("已删除记录: %v\n", e.Model)
    return nil
})
```

#### OnRecordBeforeAuthWithPassword / OnRecordAfterAuthWithPassword
```go
// 密码认证之前
app.OnRecordBeforeAuthWithPassword().Add(func(e *core.RecordAuthEvent) error {
    fmt.Printf("用户尝试登录: %s\n", e.Identity)
    return nil
})

// 密码认证之后
app.OnRecordAfterAuthWithPassword().Add(func(e *core.RecordAuthEvent) error {
    fmt.Printf("用户已登录: %v\n", e.Record)
    return nil
})
```

#### OnRecordBeforeListRequest / OnRecordAfterListRequest
```go
// 列表请求之前
app.OnRecordBeforeListRequest().Add(func(e *core.RecordListEvent) error {
    fmt.Printf("列表请求: %s\n", e.Collection.Name)
    return nil
})

// 列表请求之后
app.OnRecordAfterListRequest().Add(func(e *core.RecordListEvent) error {
    fmt.Printf("返回 %d 条记录\n", len(e.Records))
    return nil
})
```

#### OnServe
```go
// 应用启动时
app.OnServe().Add(func(e *core.ServeEvent) error {
    fmt.Println("应用正在启动...")
    return nil
})
```

#### OnTerminate
```go
// 应用终止时
app.OnTerminate().Add(func(e *core.TerminateEvent) error {
    fmt.Println("应用正在关闭...")
    return nil
})
```

### Hook 示例

#### 示例 1: 自动设置创建时间

```go
app.OnModelBeforeCreate().Add(func(e *core.ModelEvent) error {
    if record, ok := e.Model.(*core.Record); ok {
        record.SetDataValue("createdAt", time.Now().Unix())
    }
    return nil
})
```

#### 示例 2: 数据验证

```go
app.OnModelBeforeCreate().Add(func(e *core.ModelEvent) error {
    if record, ok := e.Model.(*core.Record); ok {
        title := record.Data().GetString("title")
        if len(title) < 10 {
            return errors.New("标题至少需要 10 个字符")
        }
    }
    return nil
})
```

#### 示例 3: 审计日志

```go
app.OnModelAfterCreate().Add(func(e *core.ModelEvent) error {
    logAction("create", e.Model)
    return nil
})

app.OnModelAfterUpdate().Add(func(e *core.ModelEvent) error {
    logAction("update", e.Model)
    return nil
})

app.OnModelAfterDelete().Add(func(e *core.ModelEvent) error {
    logAction("delete", e.Model)
    return nil
})

func logAction(action string, model core.Model) {
    fmt.Printf("[AUDIT] %s: %s\n", action, model.Id())
}
```

#### 示例 4: 邮件通知

```go
app.OnModelAfterCreate().Add(func(e *core.ModelEvent) error {
    if e.Collection().Name == "posts" {
        record := e.Model.(*core.Record)
        sendEmail(
            "admin@example.com",
            "新文章已创建",
            fmt.Sprintf("新文章: %s", record.Data().GetString("title")),
        )
    }
    return nil
})
```

## 2. Custom Routes（自定义路由）

PocketBase 允许你添加自定义 API 路由来处理特定请求。

### 基本路由

```go
app.OnServe().Add(func(e *core.ServeEvent) error {
    // 添加 GET 路由
    e.Router.GET("/api/hello", func(c echo.Context) error {
        return c.JSON(200, map[string]string{
            "message": "Hello, World!",
        })
    })

    // 添加 POST 路由
    e.Router.POST("/api/echo", func(c echo.Context) error {
        data := make(map[string]interface{})
        if err := c.Bind(&data); err != nil {
            return err
        }
        return c.JSON(200, data)
    })

    return nil
})
```

### 路由参数

```go
e.Router.GET("/api/users/:id", func(c echo.Context) error {
    id := c.Param("id")
    return c.JSON(200, map[string]string{
        "userId": id,
    })
})
```

### 查询参数

```go
e.Router.GET("/api/search", func(c echo.Context) error {
    query := c.QueryParam("q")
    page := c.QueryParam("page")
    return c.JSON(200, map[string]string{
        "query": query,
        "page":  page,
    })
})
```

### 认证路由

```go
e.Router.GET("/api/profile", func(c echo.Context) error {
    record, _ := c.Get("auth_record").(*core.Record)
    if record == nil {
        return c.JSON(401, map[string]string{
            "error": "未认证",
        })
    }

    return c.JSON(200, record)
}, apis.RequireAuth())
```

### 中间件

```go
// 自定义中间件
customMiddleware := func(next echo.HandlerFunc) echo.HandlerFunc {
    return func(c echo.Context) error {
        // 请求前处理
        fmt.Println("请求进入中间件")

        if err := next(c); err != nil {
            return err
        }

        // 请求后处理
        fmt.Println("请求离开中间件")
        return nil
    }
}

e.Router.GET("/api/protected", func(c echo.Context) error {
    return c.JSON(200, map[string]string{
        "message": "这是受保护的端点",
    })
}, customMiddleware, apis.RequireAuth())
```

### 与 PocketBase 集成

```go
e.Router.GET("/api/posts", func(c echo.Context) error {
    records, err := app.FindRecordsByFilter(
        "posts",
        "status = 'published'",
        "-created",
        10,
        0,
    )
    if err != nil {
        return c.JSON(500, map[string]string{
            "error": err.Error(),
        })
    }

    return c.JSON(200, records)
})
```

### 自定义路由示例

```go
app.OnServe().Add(func(e *core.ServeEvent) error {
    // 统计 API
    e.Router.GET("/api/stats", func(c echo.Context) error {
        totalUsers, _ := app.CountRecords("users")
        totalPosts, _ := app.CountRecords("posts")

        return c.JSON(200, map[string]interface{}{
            "users": totalUsers,
            "posts": totalPosts,
            "timestamp": time.Now().Unix(),
        })
    })

    // 批量操作 API
    e.Router.POST("/api/batch", func(c echo.Context) error {
        data := struct {
            IDs []string `json:"ids"`
            Action string `json:"action"`
        }{}
        c.Bind(&data)

        for _, id := range data.IDs {
            record, err := app.FindRecordById("posts", id)
            if err != nil {
                continue
            }

            switch data.Action {
            case "publish":
                record.SetDataValue("status", "published")
            case "delete":
                app.Delete(record)
            }

            app.Save(record)
        }

        return c.JSON(200, map[string]string{
            "status": "success",
        })
    })

    return nil
})
```

## 3. Go Embedding（Go 嵌入）

将 PocketBase 嵌入到自定义 Go 应用中提供了最大的灵活性。

### 基本嵌入示例

```go
package main

import (
    "log"
    "net/http"
    "github.com/pocketbase/pocketbase"
    "github.com/pocketbase/pocketbase/core"
)

func main() {
    app := pocketbase.New()

    // 添加自定义初始化
    app.OnBeforeServe().Add(func(e *core.ServeEvent) error {
        log.Println("应用即将启动")
        return nil
    })

    // 添加自定义路由
    app.OnServe().Add(func(e *core.ServeEvent) error {
        e.Router.GET("/custom", func(c echo.Context) error {
            return c.JSON(http.StatusOK, map[string]string{
                "message": "自定义路由",
            })
        })
        return nil
    })

    if err := app.Start(); err != nil {
        log.Fatal(err)
    }
}
```

### 自定义配置

```go
package main

import (
    "log"
    "github.com/pocketbase/pocketbase"
    "github.com/pocketbase/pocketbase/core"
    "github.com/pocketbase/pocketbase/plugins/config"
)

func main() {
    app := pocketbase.New()

    // 自定义配置
    app.RootCmd.Flags().StringSlice("http", []string{"0.0.0.0:8080"}, "HTTP 服务地址")

    // 使用配置插件
    config.MustRegister(app, nil)

    if err := app.Start(); err != nil {
        log.Fatal(err)
    }
}
```

### 添加自定义中间件

```go
app.OnServe().Add(func(e *core.ServeEvent) error {
    // 添加全局中间件
    e.Router.Pre(func(next echo.HandlerFunc) echo.HandlerFunc {
        return func(c echo.Context) error {
            c.Response().Header().Set("X-Custom-Header", "value")
            return next(c)
        }
    })

    return nil
})
```

### 自定义数据源

```go
app.OnDataReady().Add(func(e *core.DataReadyEvent) error {
    // 在数据准备好后执行
    log.Println("数据库已准备就绪")

    // 可以在这里执行初始数据设置
    collection, err := app.FindCollectionByNameOrId("settings")
    if err == nil {
        // 修改集合配置
    }

    return nil
})
```

## 4. JavaScript VM (JSVM)

PocketBase 包含一个 JavaScript VM，允许使用 JavaScript 编写业务逻辑。

### 启用 JSVM

JSVM 默认启用，可以通过管理面板管理 JS 脚本。

### JS Hooks 示例

#### 创建记录时

```javascript
// 在创建记录之前
routeradd("BeforeCreate", "posts", (e) => {
    const record = e.httpContext.requestInfo().data;

    // 验证标题长度
    if (record.title && record.title.length < 10) {
        throw new BadRequestError("标题太短");
    }

    // 自动设置状态
    if (!record.status) {
        record.status = "draft";
    }

    e.next();
});
```

#### 认证时

```javascript
// 在用户认证后
routeradd("AfterAuth", "users", (e) => {
    const record = e.httpContext.requestInfo().authRecord;

    // 记录登录日志
    $app.dao().createRecord("login_logs", {
        "user": record.id,
        "timestamp": Date.now() / 1000,
        "ip": e.httpContext.requestInfo().ip
    });

    e.next();
});
```

#### 自定义路由

```javascript
// 添加自定义路由
routeradd("/api/custom", (e) => {
    const data = {
        message: "来自 JavaScript 的响应",
        timestamp: Date.now()
    };

    e.json(200, data);
});
```

### JSVM 全局函数

```javascript
// 访问数据库
$app.dao().findRecordById("posts", "RECORD_ID");
$app.dao().findRecordsByFilter("posts", "status='published'");
$app.dao().createRecord("posts", {...});
$app.dao().saveRecord(record);
$app.dao().deleteRecord(record);

// 访问集合
$app.findCollectionByNameOrId("posts");
$app.saveCollection(collection);

// 发送邮件
$app.newMail({...}).send();

// 日志
console.log("信息");
console.error("错误");
```

## 5. Migrations（迁移）

迁移用于管理数据库结构的变更。

### 创建迁移文件

```go
package migrations

import (
    "github.com/pocketbase/dbx"
    "github.com/pocketbase/pocketbase/daos"
    m "github.com/pocketbase/pocketbase/migrations"
)

func init() {
    m.Register(func(db dbx.Builder) error {
        // 执行迁移
        dao := daos.New(db)

        collection := &core.Collection{
            Name: "posts",
            Type: core.CollectionTypeBase,
            Schema: core.Schema{
                Fields: []core.Field{
                    &core.TextField{
                        Name:     "title",
                        Required: true,
                    },
                    &core.TextField{
                        Name: "content",
                    },
                    &core.SelectField{
                        Name: "status",
                        Options: []string{
                            "draft",
                            "published",
                        },
                    },
                },
            },
        }

        return dao.SaveCollection(collection)
    }, func(db dbx.Builder) error {
        // 回滚迁移
        dao := daos.New(db)
        collection, _ := dao.FindCollectionByNameOrId("posts")
        return dao.DeleteCollection(collection)
    })
}
```

### 运行迁移

```bash
# 自动运行迁移
./pocketbase serve

# 手动运行迁移
./pocketbase migrate up

# 回滚迁移
./pocketbase migrate down
```

## 6. Cron Jobs（定时任务）

使用 PocketBase 的 cron 功能创建定时任务。

### 基本 Cron 任务

```go
app.OnServe().Add(func(e *core.ServeEvent) error {
    // 每小时执行一次
    e.Schedule.MustAdd("hourly-job", "0 * * * *", func() {
        log.Println("执行每小时任务")
        // 你的任务逻辑
    })

    // 每天凌晨 2 点执行
    e.Schedule.MustAdd("daily-job", "0 2 * * *", func() {
        log.Println("执行每日任务")
        // 你的任务逻辑
    })

    // 每 5 分钟执行一次
    e.Schedule.MustAdd("frequent-job", "*/5 * * * *", func() {
        log.Println("执行频繁任务")
        // 你的任务逻辑
    })

    return nil
})
```

### Cron 表达式格式

```
* * * * *
│ │ │ │ │
│ │ │ │ └─── 星期几 (0-6)
│ │ │ └───── 月份 (1-12)
│ │ └─────── 日期 (1-31)
│ └───────── 小时 (0-23)
└─────────── 分钟 (0-59)
```

## 9. 插件开发

创建可重用的 PocketBase 插件：

```go
package myplugin

import (
    "github.com/pocketbase/pocketbase/core"
)

type MyPlugin struct {
    app core.App
    config map[string]interface{}
}

func NewPlugin(config map[string]interface{}) *MyPlugin {
    return &MyPlugin{
        config: config,
    }
}

func (p *MyPlugin) Register(app core.App) error {
    p.app = app

    // 注册 Hooks
    app.OnModelAfterCreate().Add(func(e *core.ModelEvent) error {
        return p.onRecordCreated(e)
    })

    // 注册路由
    app.OnServe().Add(func(e *core.ServeEvent) error {
        e.Router.GET("/my-plugin/status", func(c echo.Context) error {
            return c.JSON(200, map[string]string{
                "status": "ok",
            })
        })
        return nil
    })

    return nil
}

func (p *MyPlugin) onRecordCreated(e *core.ModelEvent) error {
    // 处理记录创建
    return nil
}
```

```go
    "log"
    "github.com/pocketbase/pocketbase"
    "github.com/pocketbase/pocketbase/core"
)

func main() {

    // 应用启动时
    app.OnBeforeServe().Add(func(e *core.ServeEvent) error {
        log.Println("正在启动应用...")

        // 添加自定义路由
        e.Router.GET("/api/version", func(c echo.Context) error {
            return c.JSON(200, map[string]string{
                "version": "1.0.0",
            })
        })

        // 添加定时任务
        e.Schedule.MustAdd("cleanup", "0 3 * * *", func() {
            cleanupOldData(app)
        })

        return nil

    app.OnModelAfterCreate().Add(func(e *core.ModelEvent) error {

    })

    // 用户认证后
    app.OnRecordAfterAuthWithPassword().Add(func(e *core.RecordAuthEvent) error {
        log.Printf("用户已登录: %s\n", e.Record.Email())
        return nil
    })

    if err := app.Start(); err != nil {
        log.Fatal(err)
    }
}

func cleanupOldData(app core.App) {
    // 清理旧数据的逻辑
}
```
